# E-commerce Application

An easy-to-use and user-friendly ecommerce application where you can view different products that are listed, look into the each products details and add them into your cart.

## Features

- **Product Listed:** Multiple products listed on the ecommerce store.
- **Product Details:** Each product details can be viewed seperately.
- **Add to Cart:** Add you favourite products in the cart.

## Getting Started

These instructions will help you get the project up and running on your local machine for development and testing purposes.

### Pre-requisites

- Node.js version >= v18.17.0 is required

### Installation

- Clone repo and open project folder from terminal
- Run `npm install` to install all dependencies

## Running locally

In the project directory, you can run to start the project:

### `npm run dev`
